# Gambar Print Quotation

Simpan/ganti file di folder ini untuk muncul di printout Quotation:

- `logo.png`      → logo perusahaan (header). Rekomendasi tinggi ~120px, PNG transparan.
- `signature.png` → tanda tangan + stempel (blok Regards). Rekomendasi lebar ~220px, PNG transparan.

File saat ini hanyalah placeholder transparan 1x1. Timpa dengan gambar asli (nama harus sama).
Di dev (Vite) langsung ke-reload; di produksi jalankan `bench build --app crm`.
